* a, b, c, d가 주어진다
* a * c 가 b * d * 2 로 나누어 떨어지면 1 아니면 0을 출력하라.

{% endtab %}
{% tab title='BJ_15128.py' %}

```py
a, b, c, d = map(int, input().split())
area = a * c / b / d / 2
if int(area) == area:
  print(1)
else:
  print(0)
```
