* I quacked the code! 일 때까지 줄을 읽는다.
* 만약 끝 글자가 !라면 "Quack!" 아니면 "*Nod*" 를 출력하라.

{% endtab %}
{% tab title='BJ_20216.py/' %}

```py
while True:
    a = input()
    if a == "I quacked the code!":
        break
    print("*Nod*" if a[-1] == "." else "Quack!")
```
