* 첫 줄에 n, 다음 n 줄의 수들이 주어진다.
* 각각의 수들이 이전에 나온 모든 수들보다 작거나 같게 만들게 하려면 총 몇을 빼야 하는가?

{% endtab %}
{% tab title='BJ_20674.py' %}

```py
li = list(int(input()) for _ in range(int(input())))
mn, total = li[0], 0
for n in li:
  if n > mn:
    total += n -mn
  mn = min(n, mn)
print(total)
```
