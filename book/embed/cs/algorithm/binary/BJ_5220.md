* a, b 가 주이전다.
* a를 이진수로 바꿨을때 1의 개수를 2로 나눈게 b 라면 "Valid" 아니면 "Corrupt"를 출력하라.

{% endtab %}
{% tab title='BJ_5220.py' %}

```py
for _ in range(int(input())):
    a, b = input().split()
    print("Valid" if bin(int(a))[2:].count("1") % 2 == int(b) else "Corrupt")
```
