* 돌은 1개, 3개 또는 4개 가져갈 수 있다. 마지막 돌을 가져가는 사람이 게임을 지게 된다
* Find winner

{% endtab %}
{% tab title='BJ_9658.sh' %}

```sh
read n
n=$(($n%7))
nn=( SK CY SK CY SK SK SK )
echo ${nn[$n]}
```

{% endtab %}
{% tab title='BJ_9658.py' %}

```py
n = int(input())
dp = [True, False, True, False]
for i in range(n - 3):
  dp.append(not all([dp[-4], dp[-3], dp[-1]]))
print("SK" if dp[n] else "CY")
```
