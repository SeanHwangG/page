* There are a bugs from start
* Bug reproduce after b day until die (day d)
* Find total bug after N day

{% endtab %}
{% tab title="BJ_2056.py" %}

```py
import collections
a, b, d, n = map(int, input().split())
dq = collections.deque([1])
cnt = 0
for i in range(d-1):
  dq.append(0)
for i in range(n):
  cnt = (cnt + dq[a-1] - dq[b-1]) % 1000
  dq.appendleft(cnt)
  dq.pop()
print(sum(dq)%1000)
```
