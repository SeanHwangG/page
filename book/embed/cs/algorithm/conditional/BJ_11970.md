* 두 선분의 시작점과 끝점이 주어질 떄 중복 된 선분의 길이를 출력하라

{% endtab %}
{% tab title='BJ_9498.py' %}

```py
a, b = map(int, input().split())
c, d = map(int, input().split())
if a > c:
  a, b, c, d = c, d, a, b
print(b - a + d - c - max(0, min(d, b) - c))
```
