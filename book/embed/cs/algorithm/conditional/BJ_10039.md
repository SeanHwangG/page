* Print average of 5 scores, if score cannot be lower than 40

{% endtab %}
{% tab title='BJ_10039.py' %}

```py
a = max(40, int(input()))
b = max(40, int(input()))
c = max(40, int(input()))
d = max(40, int(input()))
e = max(40, int(input()))
sum = (a+b+c+d+e) // 5
print(sum)
```
