* Given Three integers, find equation using operations

{% endtab %}
{% tab title='BJ_2985.py' %}

```py
a, b, c = map(int, input().split())
if a + b == c:
  print(f"{a}+{b}={c}")
elif a - b == c:
  print(f"{a}-{b}={c}")
elif a * b == c:
  print(f"{a}*{b}={c}")
elif a // b == c:
  print(f"{a}/{b}={c}")
elif a == b + c:
  print(f"{a}={b}+{c}")
elif a == b - c:
  print(f"{a}={b}-{c}")
elif a == b * c:
  print(f"{a}={b}*{c}")
elif a == b // c:
  print(f"{a}={b}/{c}")
```
