* Check whether list has duplicates in it

{% endtab %}
{% tab title='LC_217.js' %}

```js
var containsDuplicate = function(nums) {
  return new Set(nums).size < nums.length;
};
```

{% endtab %}
{% tab title='LC_217.py' %}

```py
class Solution(object):
def containsDuplicate(self, nums):
  return len(nums) != len(set(nums))
```
