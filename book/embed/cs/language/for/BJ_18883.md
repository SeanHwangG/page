* Given N, M, output in the output format from 1 to NÃ—M

{% endtab %}
{% tab title='BJ_18883.py' %}

```py
n, m = map(int, input().split())
num = 0
for i in range(n):
  for j in range(m):
    num += 1
    if j == m - 1:
      print(num)
    else:
      print(num, end=' ')
```
