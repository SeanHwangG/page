* 두 정수 N과 F가 주어진다. 지민이는 정수 N의 가장 뒤 두 자리를 적절히 바꿔서 N을 F로 나누어 떨어지게 만들려고 한다
* 만약 가능한 것이 여러 가지이면, 뒤 두 자리를 가능하면 작게 만들려고 한다.

{% endtab %}
{% tab title='BJ_1075.py' %}

```py
a, b = int(input()), int(input())
a //= 100
for n in range(b):
  if (a * 100 + n) % b == 0:
    print(f'{n:02d}')
```
