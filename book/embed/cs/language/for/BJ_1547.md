* 세준이는 컵의 위치를 총 M번 바꿀 것이며, 컵의 위치를 바꾼 방법이 입력으로 주어진다
* 위치를 M번 바꾼 이후에 공이 들어있는 컵의 번호를 구하는 프로그램을 작성하시오.

{% endtab %}
{% tab title='BJ_1547.py' %}

```py
n = int(input())
cur = 1
for _ in range(n):
  a, b = map(int, input().split())
  if cur == a:
    cur = b
  elif cur == b:
    cur = a
print(cur)
```
